Class MStats

Manages all connections and documents in the mStats database. It's a very independent class with only minimal actual input (besides the action logging) required. Each hour it accumulates the stats into one doc. A few seconds after 00:00 UTC it collects all hours and merges them into a day doc. This day doc will also be used to update the all time doc. All hours from the last day will be deleted so there will never be more the 25 hour docs (the next hour doc might get created before the others get deleted).
